Mars Rover Kata

Clara Ameller

Explanation

  1. The symbol # is for the rovers starting position
  2. Every step done is marked by +
  3. The actuals positions are 1 and 2 for the first and second rover, respectively
  4. Every obstacle is marked by a ?

How it's done

  1. The game is based on a 10x10 grid with random obstacles, at createGrid()
  2. There are two objects: myRover1 and myRover2 with name, position, direction and marker as properties
  3. They can goForward(), goBackward(), turnLeft() and turnRight()
  4. Each of these functions call sphere() and findObstacles()
  5. sphere() checks if the new position would go off the grid. If so, it changes the position to the last/first of the grid
  6. findObstacles() checks if the new position would go on an obstacle or collide with another rover. If so, it sends a personalized message to Nasa and stops moving
  7. All this is done in the function moveRover() which takes the string with the commands, the rover itself and the grid which on is moving
